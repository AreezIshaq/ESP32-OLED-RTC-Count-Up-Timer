Instead of immediately building a complete smart alarm clock, I wanted to understand each subsystem individually.

This project focuses on learning:

RTC communication over I²C
Working with real-world time
Updating an OLED display efficiently
Organizing Arduino projects

By testing these features separately, integrating them into the final project becomes much easier and more reliable.


Future Plans

This project will become part of a larger ESP32 Smart Alarm Clock that will include:

Alarm system




OLED menu interface


Pixel animations


Motion-controlled alarm dismissal


Buzzer


Multiple alarms


EEPROM storage


Weather information
